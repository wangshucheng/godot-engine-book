func _process(delta):

    var frame_time = Engine.get_process_usec() / 1000.0
    
    if fps < 30:
        push_warning("FPS 过低：", fps)
    
    if frame_time > 33.0:
        push_warning("帧时间过长：", frame_time, "ms")

## 📈 性能基准参考

### 目标性能指标

| 平台 | 目标 FPS | Draw Calls | 物理对象 | 内存占用 |
|------|---------|-----------|---------|---------|
| PC（高端） | 144 | < 500 | < 1000 | < 2GB |
| PC（中端） | 60 | < 300 | < 500 | < 1GB |
| 主机 | 60 | < 400 | < 800 | < 3GB |
| 移动（高端） | 60 | < 150 | < 200 | < 500MB |
| 移动（中端） | 30 | < 100 | < 100 | < 300MB |
| Web | 60 | < 80 | < 50 | < 200MB |

### 优化优先级

1. **CPU 瓶颈**：代码优化 → 物理优化 → 动画优化
2. **GPU 瓶颈**：渲染优化 → UI 优化 → 材质优化
3. **内存瓶颈**：资源管理 → 对象池 → 压缩
4. **加载瓶颈**：异步加载 → 预加载 → 压缩
